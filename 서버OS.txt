* 서버 OS (Server Operating System) : 서버 컴퓨터에서 실행되는 운영체제

- 주요 서버 OS의 종류와 특징

1. Windows Server : .NET 프레임워크 등 마이크로소프트 기술과의 통합이 원활 / 보안과 관리가 잘 갖춰져있어 관리가 용이

2. Linux : 오픈 소스 운영체제, 높은 안저성과 보안성, 매우 흔히 사용, 다양한 커스터마이징 가능

3. macOS Server :  애플이 개발한 서버 운영 체제

==> 서버 OS의 선택은 웹 서비스의 용도, 트래픽, 개발 환경, 보안 등 다양한 요소 고려해야함.
