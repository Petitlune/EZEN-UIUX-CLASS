react -  hook함수

useState( ): Component의 State 변수값을 컨트롤 해주는 훅 함수
useRef( ): 특정 요소의 DOM을 컨트롤 해주는 훅 함수
useEffect( ): Component의 "생애주기"(life cycle)를 컨트롤해주는 훅 함수
 - 컴포넌트 : 생애주기 (mount > update > unmount)
1) 의존성 배열이 업테이트 || 변경되면 useEffect( ) 안에 callback 함수 실행
 => 외부 API => state => 웹브라우저 처음 켜졌을 때에는 데이터 존재 X => 사용자가 특정 버튼 클릭하면 그때 나타나게 해주는 것은 useEffect 훅을 이용한 것이다 .
