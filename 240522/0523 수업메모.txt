canvanborad 구조
1. 사용자가 input 태그를 활용해서 Text를 입력한다. 
2. 입력한  Text + 해당 Text 작성 날찌 + Text 만의 고유 id가 하나의 세트화 된 데이터가 필요.
3. 세트화 된 데이터를 담을 수 있는 자료형태 필요. 
4. 이 데이터는 어디서든 받아올 수 있도록 처리해야한다.

*목업 데이터를 만들었다.

 => ["a", "b", "c", "d", "e"]
 - 배열 안에서 특정 요소를 찾아서 인덱스 위치값을 바꿔주는 역할을 담당할 수 있는 메서드 함수를 알아야한다. (전개연산자 구문을 활용해서 기존 배열에서 새로운 값을 추가하고 기존 값을 제거하는 역학을 했었다. 그러니 값을 추가하고 제거하는 역할은 지금 상황에서는 필요하지 않음)
 - 
 => [ "c", "b", "a", "d", "e"]
