- component가 처음 마운트 되었을 때 현재 나의 위치 정보에 따른 날씨정보가 출력

=>컴포넌트 안에 어떤 특정값이 이벤트로 인해서 변경 업뎃 == useState

- 하단에 있는 도시명이 기입된 버튼 클릭시 리액트는 사용자가 어떤 버튼을 누르게되었는지를 일차적으로 인지할 수 있어야 함. => 상태가 변화되었다.  => 인지하게되었다면  해당 도시명을 가지고 API 주소에 도시명을 적용해서 해당 도시의 작용되는 날씨 정보를 가져와서 화면에 출력해주어야한다. 

- 컴포넌트의 상태 변화에 따른 어떤 명령문을 실행시키고 싶다 => useEffect
