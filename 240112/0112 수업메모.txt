 1) block Tag
    =>현재 해당 태그가 위치한 영역의 한 줄(가로=행)을 전체 사용하는 태그

    =>현재 해당 태그가 위치한 영역의 한 줄을 다른 inline 태그들과 공유해서 사용
         ex. img / a / br / button / label / input

CSS에서 가장 중요한 요소는 '선택자'
1)전체 선택자 : */ css 스타일을 적용하기에 앞서서 리셋하고자 할 때
2)태그 선택자 : html에서 작성한 tag를 직접 불러와서 적용하고자 할 때
3)아이디 선택자 : html에서 id 속성값을 사용하고 있는 태그를 선택해서 불러오고자 할 때 =>반드시 #아이디속성값 입력
   ㄴ> 단독의 유일한 값을 보유하고있는 요소에 스타일을 적용하고자 할 때 사용 
               (주의사항 - 동일한 아이디의 속성값이 존재하면 안됨)     
4)클래스 선택자 : html에서 class의 속성값을 사용하고 있는 태그를 불러오고자 할 때 => 반드시 .클래스속성값 입력
    ㄴ> 동일한 스타일을 복수의 태그 및 요서에 적용하고자 할 때 사용
