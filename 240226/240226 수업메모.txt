10. 객체!!!!
-window : 웹 브라우저 안에 있는 최상위 "객체"

 > 내장 객체 : window 기본적으로 탑재되어있는 객체
-window 자주 사용하는 프로퍼티

(앞에 윈도우는 생략하고 사용해줌)
document : 웹 브라우저에 출력될 문서 ==>html > body
innerHeight : 문서 내용의 높이
innerWidth : 문서 내용의 너비
outterHeight : 웹 브라우저 창의 높이
outterWidth : 웹 브라우저 창의 너비
screenX : 모니터의 왼쪽부터 떨어진 지점의 x좌표값
screenY : 모니터의 상단부터 떨어진 지점의 y좌표값
scrollX (=pageXOffset) : 스크롤 시 수평으로 이동한 만큼의 x좌표값
scrollY (=pageyOffset) : 스크롤 시 수직으로 이동한 만큼의 y좌표값
localStorage = 웹 브라우저에서 생성되는 데이터를 저장할 수 있도록 해주는 프로퍼티(*웹 브라우저가 종료되어도 데이터 유지)
sessionStorage = 웹 브라우저에서 생성되는 데이터를 저장할 수 있도록 해주는 프로퍼티(*새로고침/종료 데이터 리셋
location = window 객체의 현 위치를 url로 표기

- window 에서 자주 사용하는 매서드(*함수)
alert( ) : 웹 브라우저 알림창을 표시할 때
prompt( ) : 사용자로부터 입력된 텍스트를 사용하고자 할 때
focus( ) : 특정 윈도우창에 마우스 커서가 위치했을 때
open( ) : 새로운 창을 열고자 할 때
close( ) : 현재 열려있는 창을 닫고자 할 때
confirm( ) : 확인 및 취소 버튼이 있는 확인 창을 표시
print( ) : 현재 웹 브라우저 문서를 출력하고자 할 때
scroll( ) : 문서에서 특정위치로 스크롤 하고자 할 때
scrollTo( ) : 
stop( ) : 웹 문서 로딩을 멈추게 하고 싶을 때

Date( ) 프로토타입을 통해서 생성되는 인스턴스 객체가 사용할수 있는 메서드 함수 

getFullYear( ) : 연도
getMonth( ) : 월 값을 가져올 때 ( 0===1월)
getFullDate( ) : 일 값을 가져올 때
getDay( ) : 요일 값을 가져올 때 ( 0 === 일요일)
getTime( ) : 시간의 값을 가져올 때 ( 기준은 1970년 1월 1일부터 현시점까지의 시간 / 밀리초의 시간 개념으로 값을 가져옴 )
getHours( ) : '시' 값 
getMinutes( ) : '분' 값 
getSeconds( ) : '초' 값 

setFullYear( ) : 연도 설정
setMonth( ) : 월 값을 설정 할 때 ( 0===1월)
setFullDate( ) : 일 값을 설정 할 때
setDay( ) : 요일 값을 설정 할 때 ( 0 === 일요일)
setTime( ) : 시간의 값을 설정 할 때 ( 기준은 1970년 1월 1일부터 현시점까지의 시간 / 밀리초의 시간 개념으로 값을 설정함 )
===>1밀리초 =1/1000초
===>1초 = 1000밀리초
===> 1분 =60초(60*1000 =60000밀리초)
===> 1시간 =60분(60*60*1000 =3,600,000밀리초)
===> 1일 =24시
setHours( ) : '시' 값 설정 
setMinutes( ) : '분' 값 설정
setSeconds( ) : '초' 값 설정
